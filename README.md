# Opportunity Atlas

This project investigates how access to information, mentorship, and socioeconomic conditions shape educational opportunity among students.

The goal is to understand why equally capable students often experience different educational outcomes.

## Methods
- Data collection (student surveys)
- Python-based data analysis
- Visualization and basic machine learning

## Current Stage
- Designing survey and defining variables
